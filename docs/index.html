⚠️Demo Limitations

Important: HAtui is designed for people who already have a Home Assistant smart home setup. Without smart devices, you can't experience the full magic of controlling your lights, switches, and sensors.

However, I understand you might want to try it out before setting up your own smart home...

Test Environment Available

I've set up a hosted Home Assistant instance in the cloud with simulated devices for testing purposes. You can control virtual switches and see sensors, but you won't see any physical feedback (no actual devices turning on/off).

Think of it as a "sandbox mode" - all the interface functionality works, but the magic of controlling real devices is missing.

Features

Multi-Dashboard Support

Create and manage multiple custom dashboards for different rooms or purposes.

Real-time Control

Toggle lights, adjust brightness, monitor sensors with instant feedback.

Keyboard Navigation

Full keyboard-driven interface - no mouse required, perfect for terminal enthusiasts.

Entity Management

Add, remove, and rename entities in edit mode. Drag and drop to rearrange.

Brightness Staging

Smooth brightness control with staging to prevent API flooding.

Auto-Configuration

Creates default config on first run. No complex setup required.

Controls

Navigation

  • Arrow Keys: Navigate entities
  • Space: Toggle entity
  • Ctrl+↑↓: Adjust brightness
  • Ctrl+←→: Switch dashboards
  • r: Refresh entities
  • q: Quit

Edit Mode (press 'e')

  • a: Add new entity
  • Delete: Remove entity
  • Enter: Pick/Drop entity
  • n: Edit entity name
  • d: Dashboard management
  • Escape: Exit edit mode

Test Environment Notes

Server Stability: The test server might occasionally go down due to resource limits or high traffic.

No Visual Feedback: You can toggle switches, but won't see any physical changes since they're virtual devices.

Limited Devices: Only a few test entities are available compared to a real smart home setup.

For Your Own Smart Home

To use HAtui with your actual Home Assistant setup:

  1. Get your Home Assistant URL and long-lived access token
  2. Create a .env file with your credentials: 
    HA_URL=http://your-ha-instance:8123
HA_TOKEN=your_long_lived_access_token
  3. Run HAtui and enjoy controlling your real smart devices!
